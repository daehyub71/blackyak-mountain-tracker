## 매핑 제외 산 목록

### 1. 잘못된 매핑 (수동 확인 필요)

다음 산들은 MNT_CODE에 동명이산이 있어 자동 매핑이 불가합니다:

- 계룡산 (충청남도)
- 무등산 (광주광역시)
- 축령산(장성) (전라남도)
- 팔공산 (대구광역시)
- 가야산(경상) (경상남도)
- 가야산(충남) (충청남도)
- 백암산 (전라북도)
- 태백산 (강원도)

### 2. 국립공원 산 (별도 API 필요)

다음 산들은 MNT_CODE에 없으며, 국립공원공단 API를 사용해야 합니다:

- 설악산, 지리산, 한라산
- 내장산, 덕유산, 소백산, 속리산
- 오대산 (노인봉, 비로봉)
- 월악산, 월출산, 치악산
- 두륜산, 내변산(부안)

### 3. 기타 미매칭

- 가리산(홍천) - MNT_CODE에 없음
- 동악산(곡성) - 둘레길만 등록
- 선운산 - "선운산_도솔봉"으로 등록
- 용봉산(홍성) - MNT_CODE에 없음
- 장안산 - "장안산_상봉"으로 등록
- 조계산 - "조계산_장군봉"으로 등록

## 산코드 활용 방법

산림청 API에서 GPX 파일을 다운로드하려면:

```
https://www.forest.go.kr/newkfsweb/cop/openapi/openApiGpxDown.do?mntCd={산코드}
```

예시: 가리왕산 GPX 다운로드
```
https://www.forest.go.kr/newkfsweb/cop/openapi/openApiGpxDown.do?mntCd=427700101
```
