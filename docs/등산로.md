### 1. 등산로 데이터 수집 크롤러 (API 활용)

이 코드는 산림청 API를 호출하여 **"특정 산(예: 설악산)의 등산로 구간 정보(거리, 난이도, 시간)"**를 텍스트로 긁어오는 역할을 합니다.

**준비물:**

* 공공데이터포털 [산림청_등산로정보서비스] 활용신청 후 받은 **Encoding 인증키**
* `requests` 라이브러리 설치 (`pip install requests`)

```python
import requests
import xml.etree.ElementTree as ET
import pandas as pd

def get_hiking_trail_data(mountain_name, api_key):
    # 산림청 등산로 구간 정보 API 엔드포인트
    url = 'http://apis.data.go.kr/1400000/service/cultureInfoService2/mntInfoOpenAPI2'
    
    params = {
        'serviceKey': api_key,
        'searchWrd': mountain_name, # 검색할 산 이름 (예: 설악산)
        'numOfRows': '100',         # 한 번에 가져올 데이터 수
        'pageNo': '1'
    }

    try:
        response = requests.get(url, params=params)
        
        # XML 응답 파싱
        if response.status_code == 200:
            root = ET.fromstring(response.content)
            items = root.findall('.//item')
            
            data_list = []
            for item in items:
                # 필요한 정보만 쏙쏙 뽑기
                trail_info = {
                    '산이름': item.findtext('mntnm'),      # 산 이름
                    '높이': item.findtext('mntheight'),    # 해발고도
                    '상세정보': item.findtext('mntdtails'), # 코스 설명
                    '위치': item.findtext('rnaddr'),        # 주소
                }
                data_list.append(trail_info)
            
            return pd.DataFrame(data_list)
        else:
            print(f"Error: {response.status_code}")
            return None

    except Exception as e:
        print(f"API 호출 중 오류 발생: {e}")
        return None

# --- 실행부 ---
# 여기에 공공데이터포털에서 받은 본인의 '인증키(Encoding)'를 넣으세요
MY_API_KEY = "여기에_인증키_붙여넣기" 

# 테스트: 설악산 정보 가져오기
df = get_hiking_trail_data("설악산", MY_API_KEY)

if df is not None:
    print(df.head())
    # 엑셀로 저장하려면: df.to_excel("설악산_정보.xlsx")

```

### 2. (핵심) 지도에 선 그리기용 데이터 처리 (GeoPandas)

사실 질문자님이 진짜 필요한 건 위의 텍스트 정보보다, 지도 위에 선을 그릴 수 있는 **좌표 데이터(LineString)**일 것입니다.

이건 API로 하나씩 부르는 것보다, 공공데이터포털에서 **Shapefile(.shp) 전체를 다운로드** 받아서 파이썬으로 여는 게 정석입니다.

**준비물:**

* 데이터: 공공데이터포털 '산림청_등산로 정보(공간정보)' 다운로드
* 라이브러리: `pip install geopandas matplotlib`

```python
import geopandas as gpd
import matplotlib.pyplot as plt

# 1. 다운로드 받은 등산로 Shapefile 읽기 (파일명은 예시)
# 파일 경로에 한글이 있으면 에러가 날 수 있으니 영문 경로 권장
shp_path = "./korea_trails_data/fr_road.shp" 

try:
    gdf = gpd.read_file(shp_path, encoding='euc-kr') # 한글 깨짐 방지

    # 2. '설악산' 데이터만 필터링 (컬럼명은 데이터 명세서 확인 필요, 보통 MNT_NM 등)
    # 데이터 컬럼명을 모를 땐 print(gdf.columns)로 확인
    seorak_trails = gdf[gdf['MNT_NM'] == '설악산']

    # 3. 데이터 확인 (구간별 거리, 난이도 등이 들어있음)
    print(seorak_trails[['MNT_NM', 'SEC_LEN', 'CAT_NAM']].head()) 
    # SEC_LEN: 구간거리, CAT_NAM: 난이도(상/중/하)

    # 4. 간단하게 시각화 (제대로 로드됐는지 확인용)
    seorak_trails.plot()
    plt.title("Seoraksan Hiking Trails")
    plt.show()

except Exception as e:
    print(f"파일 로드 실패: {e}")

```
